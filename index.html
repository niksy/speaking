bower.io

Kako instaliram Bower?


							npm install -g bower

Ugh, konzola…

Kako instaliram paket?


							bower install <paket>

Paketi se instaliraju u
"bower_components" direktorij


							bower install <paket>#<verzija>
						

							bower install <ime>=<paket>#<verzija>

--save

Ne sviđa mi se
instalacijski direktorij…

.bowerrc

{
  "directory": "direktorij/mog/projekta"
}

Kako brišem paket?


						bower uninstall <paket>

Kako nadograđujem paket?


						bower update <paket>
